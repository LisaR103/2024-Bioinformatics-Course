SciPy¶

Scipy is a colletion of mathematical algorithms built on NumPy

The subpackages need to be imported separately, e.g.

from scipy import linalg, optimize

functions are then used with .notation

linalg.inv(matrix)

A list of subpackages is in the powerpoint, SciPy has its own website: https://scipy.org

Matplotlib¶

A comprehensive library for creating static, animated, and interactive visualizations in Python.
To use in Python the submodule pyplot is needed:

import matplotlib.pyplot as plt

some plottypes, all are called with plt.name():

  • plot (scatter or line plots)
  • bar (vertical bars)
  • errorbar (scatter or line plots with errorbars)
  • barh (horizontal bars)
  • hist (histograms)
  • contour (contour plots?
  • imshow (heatmap)
  • pie
  • plot_surface
  • bar3d

Modifications similar to ggplot in having them added underneath the call of the plot

  • tilte
  • xlabel
  • ylabel
  • xlim
  • ylim
  • xticks
  • yticks
  • grid
  • legend

Write a python script that creates the following three different plots with the help of Matplotlib:

  • a Lineplot with at least 10 datapoints
  • a Scatterplot with at least 500 datapoints
  • a bar diagram with at least 15 bars
